一个完整的全栈项目应该是怎么样的？

发表于2025-07-19|更新于2025-08-03
|浏览量:

一、概述

  • vue3
  • spring boot3
  • mysql
  • nginx
  • redis
  • jenkins

二、前端部分

  • 使用 Vue 3 + ts 作为前端框架
  • 使用 Vue Router 进行路由管理
  • 使用 pinia 作为状态管理
  • 使用 axios 进行 HTTP 请求
  • 使用 Element Plus 作为 UI 组件库
  • 使用 Vite 作为构建工具
  • 使用 ESlint 和 Prettier 进行代码规范和格式化
  • 配置插件用于组件的自动导入和项目的轻量化

三、后端部分

  • 使用 Spring Boot 作为后端框架
  • 使用 MySQL 作为数据库
  • 使用 Redis 作为缓存
  • 使用 JWT 进行身份验证
  • 使用 Swagger 进行接口文档

四、运维部分

  • 使用 Nginx 作为反向代理
  • 使用 Jenkins 进行持续集成和部署

五、总结

施工中🚧
